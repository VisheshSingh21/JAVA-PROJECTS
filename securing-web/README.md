<h1>🔐 Securing Web Application</h1>
<p><b>Spring Boot project demonstrating authentication and authorization using Spring Security.</b></p>

<div class="card">
<h2>📌 About the Project</h2>
<p>
This project is built using Spring Boot to demonstrate how web applications can be secured using authentication and authorization mechanisms.
It includes a login system, protected routes, and role-based access control concepts.
</p>
</div>

<div class="card">
<h2>🎯 What I Learned</h2>
<ul>
<li>How Spring Security works internally</li>
<li>Authentication vs Authorization</li>
<li>Securing endpoints using configuration</li>
<li>Creating custom login pages</li>
<li>Handling user sessions securely</li>
</ul>
</div>
